# MetabolicPotential
Finite element analysis models used to implement the free-energy based approach for calculating the metabolic potential as outlined in "Mechanical regulation of cellular energy metabolism in cancer microenvironments", bioRxiv 2024.04.30.591879; doi: https://doi.org/10.1101/2024.04.30.591879

Please cite our paper if this code is used or modified for further use.

This package is supported for macOS and windows. The package has been tested on the following systems:

• macOS Monterey 12.4, Processor: 2.3 GHz 8-Core Intel i9, RAM: 32 GB
• Windows 11 Home, Processor: AMD Ryzen 5 5500U Hexa-Core processor, RAM: 16 GB

Models are built in COMSOL Multiphysics 6.3. Install COMSOL Multiphysics (https://www.comsol.com/) along with the Structural Mechanics Module. All simulation results used to generate figures in the manuscript are obtained from the files.
